О проекте

Цель: разработать собственную файловую базу данных на C#, которая хранит объекты в JSON-файлах и поддерживает все основные операции с данными, без использования готовых СУБД.

Задачи: спроектировать архитектуру, реализовать CRUD-операции, создать консольное приложение для демонстрации, оформить документацию и веб-сайт.

Архитектура классов

База данных спроектирована так, чтобы работать с любыми типами объектов. Все хранимые классы реализуют единый интерфейс IEntity, а ядро DataBase оперирует ими через обобщённые методы.

Структура хранения данных

Каждая таблица — это папка на диске. Каждая запись — отдельный JSON-файл. Имя папки соответствует типу объекта, имя файла — уникальному идентификатору.

Как происходит сохранение: метод Save

Метод Save<T> работает с любым типом, реализующим IEntity. Определяет папку по имени типа, генерирует GUID для новых записей, сериализует объект в JSON и сохраняет в файл.

public void Save<T>(T entity) where T : IEntity
{
    string tablePath = Path.Combine(_rootPath, typeof(T).Name);
    Directory.CreateDirectory(tablePath);

    if (entity.Id == Guid.Empty)
        entity.Id = Guid.NewGuid();

    string filePath = Path.Combine(tablePath, $"{entity.Id}.json");
    string json = JsonSerializer.Serialize(entity);
    File.WriteAllText(filePath, json);
}
